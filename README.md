# 1st Annual Bellairs Workshop on Experimental & Computational Neuroscience


## What is neuronify
Neuronify is an educational neural network app. It allows you to create and explore neural networks in a plug-and-play simulation environment. You can download the app on a computer from this website: https://ovilab.net/neuronify/

## Elements used in the workshop from neuronify
### Neuron activator
![alt text](https://github.com/mengyixu/Bellairs_Workshop_2026/blob/main/example/elements_from_neuronify/input.png "Neuron activator")

When it is connected to a neuron, it delivers a constant electrical current into the cell. The strength of the current can be adjusted.

### Excitatory neuron
![alt text](https://github.com/mengyixu/Bellairs_Workshop_2026/blob/main/example/elements_from_neuronify/excitatory_neuron.png "excitatory neuron")

An excitatory neuron is a cell that **_increases_** the chance that the next cell will send a signal. 

It acts like a **_green light_**, encouraging the next neuron to “fire” or pass the message along.


### Inhibitory neuron
![alt text](https://github.com/mengyixu/Bellairs_Workshop_2026/blob/main/example/elements_from_neuronify/inhibitory_neuron.png "inhibitory neuron")

An inhibitory neuron is a cell that **_decreases_** the chance that the next cell will send a signal.

It acts like a **_red light_**, slowing down or stopping the message.


### Sensor
![alt text](https://github.com/mengyixu/Bellairs_Workshop_2026/blob/main/example/elements_from_neuronify/output.png "Sensor")

It can be used to measure the electrical activity (membrane potential) of one or more neurons.
