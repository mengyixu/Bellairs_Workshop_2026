# 1st Annual Bellairs Workshop on Experimental & Computational Neuroscience


### What is neuronify
Neuronify is an educational neural network app. It allows you to create and explore neural networks in a plug-and-play simulation environment. You can download the app on a computer from this website: https://ovilab.net/neuronify/
